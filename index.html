Cryptos

Demo Github

  • Cryptos is a mobile app for tracking cryptocurrency prices and trends.

  • The Cryptos frontend is built with React Native and Redux.

  • The backend utilizes a simple Express server and stores data via an mLab-hosted Mongo Database.

  • The backend stores historical pricing data on the most popular coins. The data is displayed in graphs within the application.

BoomBlock

Live Github

  • Boomblock is an original interactive audio player.

  • It is built with Javascript, the Three.JS library, and the Web Audio API.

  • Boomblock runs on a tempo-detection algorithm that scans mp3-derived PCM data for peaks, generates a BPM, and then renders the 3D world to the beat.

  • All music is original.
